# Errors

The tronalddump api uses the following error codes:

Error | Meaning
---------- | -------
400 | Bad Request
403 | Forbidden
404 | Not Found
405 | Method Not Allowed
406 | Not Acceptable
410 | Gone
418 | I'm a teapot
429 | Too Many Requests
500 | Internal Server Error
503 | Service Unavailable
